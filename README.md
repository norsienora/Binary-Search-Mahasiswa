# Pencarian Data Mahasiswa Berdasarkan NIM

Aplikasi ini dibuat untuk memenuhi tugas mata kuliah Analisis Kompleksitas Algoritma.

## Topik
Pencarian Data Mahasiswa Berdasarkan Nomor Induk Mahasiswa (NIM) Menggunakan Algoritma Binary Search dengan Pendekatan Divide and Conquer Secara Iteratif dan Rekursif.

## Fitur
- Pencarian data mahasiswa berdasarkan NIM
- Perbandingan Binary Search iteratif dan rekursif
- Visualisasi grafik waktu eksekusi
- GUI berbasis Python (Tkinter)
